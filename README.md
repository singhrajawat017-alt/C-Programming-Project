C PROJECT - MOVIE TICKET BOOKING SYSTEM
Movie Ticket Booking System (Theater Management)
• A console-based movie ticket booking system for managing a single theater.
• Users can book tickets for 4 different movies available, with dynamic pricing depending on seat rows: Rs 150/ 200 /250）.
• The hall is represented as a 10x10 seating matrix, clearly showing which seats are booked or available.
• All available seats are shown, including row number, seat number, and price.
• Booked tickets can be canceled, and the system automatically calculates and shows the refund amount.
• It monitors the revenue separately for each movie, hence the output would be clear regarding earnings.
• Each reservation is written to a file called bookings.txt, including customer's information.
• Day-to-day revenue details are kept in daily_records.txt for record purposes.
• Seating is categorized according to comfort and price:
• 1-3 front rows
• Rows middle: 4-7
• Back rows: 8-10
• Every booking contains customer information like the customer's name and contact phone number.
• A mock payment link generation occurs after a successful booking to simulate the experience of the transaction. i
